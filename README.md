#PBlogger

A bloggging API with the following features implemented:

Basic CRUD

Post(id,title,and article)

User Auth/Auth(using JWT)

Pagination

Sorting

Filter by tags

Search by title

Validation(using joi)

Logging(usng Morgan)

Error handling

Testing(using jest/supertest)

Documentation(using Postman && SwaggerHub)
